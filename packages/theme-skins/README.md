# @resonate/theme-skins

## Installation

```sh
$ npm i --save @resonate/theme-skins
```

## Usage

```javascript

const html = require('choo/html')
const { background } = require('@resonate/theme-skins')

const el = html`<div class=${background}></div>`

document.body.appendChild(el)

```
